# 🤖 Pumba

A multifunctional Telegram bot in Ukrainian, designed for convenient interaction with friends and community.

## 📋 About the Project

Pumba (telegram: @PumbaBoarBot) is a personal project developed with the needs of my friends and other users in mind. The bot offers a wide range of useful features for everyday use.

## 🚀 Quick Start

### Prerequisites

- Python 3.x
- Telegram Bot Token

### Installation and Launch

1. **Set up environment variables**
   ```bash
   cp .env.sample .env
   ```

2. **Fill in the `.env` file**
   
   Open the `.env` file and enter the required data in all configuration fields.

3. **Run the bot**
   ```bash
   python python/main/main.py
   ```
